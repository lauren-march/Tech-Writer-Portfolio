# POST /spaces

Use the `POST` method to add a new `/spaces` resource.

## Create a space

Create a new coliving or coworking space.

### Endpoint

```shell
POST /spaces
```

### Request body

```json
{
  "name": "string",
  "location": "string",
  "pricePerHour": number,
  "capacity": number,
  "description": "string"
}
```

#### Example

```json
{
  "name": "Sunset Studio",
  "location": "Barcelona, Spain",
  "pricePerHour": 6,
  "capacity": 15,
  "description": "Bright coworking space with Mediterranean views and creative atmosphere."
}
```

### Parameters

None.

### Responses

| HTTP Code | Description | Schema |
|-----------|-------------|--------|
| 201 | Space created successfully | `spaces` object |
| 404 | Space not found | `error` object |
| 400 | Bad request - invalid data | `error` object |

### Example request

#### cURL

```bash
curl -X POST http://localhost:3000/spaces \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Sunset Studio",
    "location": "Barcelona, Spain",
    "pricePerHour": 6,
    "capacity": 15,
    "description": "Bright coworking space with Mediterranean views and creative atmosphere."
  }'
```

#### Postman

1. Set method to `POST`
1. Enter URL: `http://localhost:3000/spaces`
1. Go to **Body** tab
1. Select **raw** and choose **JSON** from dropdown
1. Paste the request body JSON:

    ```json
    {
    "name": "Sunset Studio",
    "location": "Barcelona, Spain",
    "pricePerHour": 6,
    "capacity": 15,
    "description": "Bright coworking space with Mediterranean views and creative atmosphere."
    }
    ```

1. Click **Send**

    ![](../media/post-postman-example.png)

### Example response

```json
{
  "id": 5,
  "name": "Sunset Studio",
  "location": "Barcelona, Spain",
  "pricePerHour": 6,
  "capacity": 15,
  "description": "Bright coworking space with Mediterranean views and creative atmosphere."
}
```

### Notes

*The `id` field is automatically generated by json-server
*All fields except `id` should be included in the request body
